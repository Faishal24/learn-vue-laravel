# Learn Vue 3
A simple chat app built with Vue 3, demonstrating the use of:
- Template Syntax
- Reactivity
- Computed Properties
- Class Bindings
- Conditional Rendering
- List Rendering
- Event Handling
- Form Input Bindings
- Watchers
- Template Refs
- Components
- Lifecycle Hooks
- Props
- Events
- Slots
- Composables
- Transitions / Transition Groups
- KeepAlive
- Async Components
